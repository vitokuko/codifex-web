Plateforme

Cette plateforme a ete mise en place pour faciliter la gestion des codications des etudiants de l'ecole superieure polytchnique de Dakar. Les deux grandes fonctionnalites de l'application sont :

  • Reserver
  • Codifier

Codifier

A ce niveau, vous pouvez choisir la chambre dans laquelle vous voulez codifier. En effet, seules les chambres ayant de positions libres pouront etre choisies. Notons que les chambres auront au maximun 4 etudiants occupants les positions de 1 a 4.
Pour codifier dans une chambre, il faut renseigner :

  • Le pavillon
  • Le numero de l'etage
  • Le numero de couloir
  • Le numero de la chambre
  • La position a occupee

Vous pouvez aussi codifier directement lorsque vous avez deja reserver une chambre. Dans ce cas, seule une verification des parametres de reservation est necessaire avant de valider la codification. A cette occasion, pous pouvez changer de position s'il y'a au moins une position libre.

Vous pouvez choisr parmis les chambres suggerees. La selection de ces chambres est base sur les aspects et les relations que vous pouvez avoir evec vos colocataires. Ces criteres peuvent etre entre autre :

  • Le sexe
  • Le departement
  • L'option
  • Le niveau
  • Le cycle
  • L'annee d'etude
  • l'age

Reserver

A ce niveau pous pouvez reserver une chambre en attendant les codifications. Pour reserver une chambre, il faur reseigner :

  • Le pavillon
  • Le numero de l'etage
  • Le numero de couloir
  • Le numero de la chambre
  • La position a occupee

Statistiques

A ce niveau, vous aurez toutes les informations concernant les pavillons. Pour chaque pavillon, vous pouvez savoir :

  • Le nombre de chambre total

Parametres

A ce niveau, vous pouvez modifier toutes les informations vous concernant. Ces dernieres peuvent etre personnelles commes academique (sur votre formation).

  • Nom et Prenom
  • Mot de passe
  • Photo de profil
  • Departement
  • Option
  • Cycle
  • Annee d'etude
